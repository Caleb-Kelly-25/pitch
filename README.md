# Pitch
This repository holds the code implementing a multiplayer pitch card game.

## Dev Instructions
To test locally, run the command "docker compose up --build" from the root.

Then, navigate to the /client folder, open an additional terminal, and run "npm run dev".

Then go to your browser and enter the address localhost:5173/signup


Note that you will have to have Docker Desktop installed and running on your computer before this will work.
